Credit Card Churn Analysis

The cardinal determinant of success in the consumer banking credit card industry is the churn rate, which poses a fundamental challenge to banks. Credit card churn entails the loss of a customer, resulting in the forfeiture of future profits and the operating costs incurred in acquiring that customer.

The credit card customer churn rate represents the percentage of a bank's customers who cease using the bank's services. Consequently, comprehending the characteristics of customers who churn or remain loyal is crucial for banks to customize their marketing strategies, modify their services, or introduce new ones for optimal success.

Partition-based clustering, specifically K-Means clustering, was selected for our analysis due to its superior efficiency when working with large datasets compared to hierarchical clustering methods (Jain, 1999), (Hastie, 2009). To ensure that the chosen number of clusters is optimal for our dataset, we employed the elbow method, which is a commonly used technique for selecting the number of clusters in K-Means clustering (Kassambara, 2017). Additionally, we conducted feature importance analysis to identify the most significant variables that contribute to the clustering process, allowing us to eliminate any variables that do not significantly affect clustering outcomes (Saitta, 2015).

Variable Selection

The K-Means clustering algorithm relies on measuring distances between data points in a multi-dimensional space. Consequently, using non-numeric variables like categorical variables or text data may pose a challenge because the K-Means clustering algorithm calculates the Euclidean distance between data points. As such, utilizing non-numeric variables in K-Means clustering may produce inaccurate distances and clustering outcomes.

Selecting Number of Clusters

To select the number of clusters for the optimal configuration, we use the elbow method, which is a widely accepted technique to derive the optimal number of clusters in clustering analysis. The elbow method entails running the K-Means clustering algorithm for a range of cluster values, typically from 1 to 10, and computing the sum of squared distances from each data point to its assigned centre (known as distortions).
We then plot the distortions against the number of clusters, resulting in a graph resembling an arm shape. The elbow point, which marks the point of inflection on the curve, is considered the optimal number of clusters. By identifying the elbow point, we can find the optimal cluster number and proceed with further analysis accordingly.

Figure 14: Elbow Method to Determine Optimal K Clusters

The elbow point is regarded as the best number of clusters because it indicates the stage at which the improvement in clustering performance becomes insufficient to justify the added complexity of using more clusters. In simpler terms, the elbow method assists us in identifying the point of diminishing returns, at which further cluster additions do not considerably enhance the quality of the clustering outcomes. From the results, 4 is the optimal number of clusters for our study.

7. Predictive Modelling

From the cluster analysis conducted, a predictive model will be able to help identify each customer’s cluster’s probability of churning.
The team further looked into the methods of predicting whether the customers will belong to which of the aforementioned clusters such that credit card issuers can implement different strategies on the customers at earlier stages to reduce the churn possibilities. Several methods are being considered such as (1) Logistic Regression (2) Random Forest (3) Support Vector Machine

Random Forest was selected due to its ability of handling the variation of features, not limited to only linearly correlated variables and it is less prone to overfitting. It can be used for classification and regression tasks. In addition, it is called "random" because it creates many decision trees by randomly selecting subsets of the data and subsets of the features for each tree. The algorithm predicts the label of a new data point, where each decision tree in the forest independently predicts the label based on the features of that data point, and the final prediction is based on a majority vote or average of the predictions from all the trees. It is also known to be able to handle imbalanced data, where one class of the target variable has significantly more samples than another class.
During the training process, Random Forest uses a technique called "bagging" (bootstrap aggregation), which involves randomly sampling the training data with replacement to create multiple subsets of data for each decision tree. By doing so, Random Forest reduces the variance in the model and makes it less prone to overfitting.
Additionally, Random Forest uses a technique called "random subspace method" to randomly select a subset of features for each decision tree. This technique helps to reduce the correlation between the decision trees and increases the diversity in the ensemble, which can lead to better performance.

Model Evaluation

To design a prediction model, the dataset was divided into training and testing datasets, with cut off percentages of 75% and 25% respectively. The performance of the predictive model can be evaluated by using classification parameter variables, such as: (1) Accuracy, (2) Area under curve (AUC), (3) Specificity, (4) Recall, (5) Precision, (6) F1-score. To find the performance metrics, a confusion matrix is generated first using the output of the classification results. The confusion matrix contains four parts: true positive (TP), true negative (TN), false positive (FP), and false negative (FN). The definition of the confusion matrix are:
• True positives (TP): Number of churn customers correctly predicted as true churn.
• True negatives (TN): Number of non-churn customers correctly predicted as non-churn.
• False positives (FP): Number of churn customers incorrectly predicted as non-churn.
• False negatives (FN): Number of non-churn customers incorrectly predicted as churn.

Targeted Marketing

If the target customer group consists solely of females, it may be advantageous to tailor the marketing strategy specifically for this gender. By focusing on female-specific needs, preferences, and behaviors, the marketing message can resonate more strongly with the target audience, leading to increased engagement and customer loyalty. For example, if the product or service being marketed is in the beauty industry, highlighting the product's benefits for a woman's skin type or hair texture can help to generate interest and motivate the target audience to make a purchase.
Tailoring marketing efforts for a specific gender also allows for messaging that is more targeted and personalized. By understanding the unique needs and preferences of the target audience, marketers can create content that speaks directly to them and addresses their specific concerns. This approach can help to build trust and establish a connection with the target audience, leading to increased engagement and customer satisfaction.

Increase Credit Limit

Offering flexibility to increase credit limit can be an effective strategy to encourage spending among customers. By allowing customers to have more purchasing power, they may be more likely to make larger transactions and/or make purchases more frequently. The flexibility to increase credit limit can also help to build customer loyalty by demonstrating trust in the customer's ability to manage their finances responsibly. This can help to foster a positive relationship between the customer and the credit provider, leading to increased customer satisfaction and retention.

Increase Relations

When a customer uses a banking product, it creates an opportunity for the credit card issuer to offer additional products or services that could meet the customer's needs and enhance their overall banking experience. By offering a range of products and services, the credit card issuer can deepen their relationship with the customer and encourage them to remain within their ecosystem, rather than switching to a competitor. For example, if a customer has a credit card with a bank, the bank could market other products such as personal loans, mortgages, savings accounts, or investment products to that customer. This approach can be particularly effective if the additional products are tailored to the customer's needs and preferences, as it can increase the likelihood that the customer will see value in the offerings and choose to stay with the bank.
Marketing other banking products can also be a way for the credit card issuer to generate additional revenue streams from their existing customer base. Cross-selling products to existing customers is generally less costly than acquiring new customers, as the credit card issuer already has a relationship with the customer and has access to their financial data and preferences.
Overall, marketing other banking products to existing customers can be a strategic way to deepen customer relationships, increase revenue, and build a loyal customer base.

9. Closed Looped Solution

A closed-looped ecosystem would be required in order validate the solution models that the team has proposed.
The following considerations are key to creating a closed loop solution:
(1) Update the customer groups for existing customers
We can re-run the prediction model on existing customers to observe if any customers have switched groups after implementing the recommendations. Understanding the customers who have changed groups would give us insight on the effectiveness of our campaign.
(2) Compare ‘before and after’ credit card data for existing customers
For existing customers, we should continue to collect data on their spending habits as per what has been collected in the dataset. The customer’s credit card spending data should be compared before and after the recommendations are implemented. This allows us to conclude whether the recommendation influence customer behaviour. They should be regularly monitored, so adjustments to credit card incentives could be made when necessary. These will give us further insight into how much resources we can expect to allocate to encourage spending and/or lower churn rates or make adjustments to our marketing incentives to better appeal to the customers. By regularly monitoring and refining the predictive model's performance, effective strategies can be developed to retain customers while optimizing resource utilization.  

10. Future Work

Exploring Other Clustering Models

In this project, K-Means clustering was used to segment data. Future work could involve exploring other clustering models that can handle categorical data as well. One possible option would be to explore Neural Networks, Agglomerative Clustering, and Gaussian Mixture Models. By considering a range of clustering models, the project can identify the most appropriate technique for segmenting the data and gain new insights into the underlying patterns and trends in the data.

Adjusting the Variables Used in the Predictive Model

In future work, the variables used to develop the predictive model can also be varied to identify the best independent variables that would affect the accuracy of the model.
(1) Reduce the number of variables sampled at each split
We could reduce the number of variables considered for each split to introduce more randomness into the model. This could be carried out whenever a split is created in a tree, where a subset of variables are taken and only those variables are considered to be the variable to be split on. If we consider a smaller subset of variables at each split, the trees are less likely to look the same because it is unlikely that the same variable is available for consideration at each split.
(2) Rank features based on importance before forwarding to the machine learning model
Based on the output from the feature importance study conducted, the independent variables could be divided into important, marginal, and unimportant. Variables considered to be marginal or unimportant could also be omitted from the predictive model. This may help to reduce noise from the predictive model, improving the performance of the model.

Collecting Additional Variables

In addition, analyses to further understand the optimal independent variables are required to build a more robust, more accurate, faster, less complicated prediction model. Such a study should cover a larger dataset with additional variables such as:
(1) More attrited customer data count
The count of attrited customer instance is only 16% in this current dataset. By increasing the data collection of attrited customer, the dataset will be more balance, allowing the model to cluster them better and allow the credit card issuers to capture more information and insights regarding the reasons of attrition.
(2) Customer Transaction History and Communication Preference
Analysis on the customer’s spending patterns, transaction history and communication preferences, will provide insights to the reasons behind their churn, and allow better development of the targeted marketing strategy.
(3) Geographical data – residential location, work location
To better understand if any segments or metrics are separated spatially. This would provide additional information for the company to plan different strategic methods targeting different segments.
(4) Demographic data – employment information
To provide better clarification about the user, credit card issuers could also assess the users’ employment information such as his job title, and employer name. This would provide insights into the user’s creditworthiness as well as financial and job stability.
